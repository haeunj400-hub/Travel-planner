# 국내 여행 플래너

공공데이터포털 TourAPI를 활용한 국내 여행 플래너 웹 서비스입니다.

---

## 주요 기능

- 지역별 관광지 조회
- 관광지 검색
- 관광지 상세 페이지
- 찜 기능(localStorage)
- 찜 목록 페이지

---

## 사용 기술

### Frontend
- React
- Vite
- Tailwind CSS
- React Router

### Backend
- Node.js
- Express

### DevOps
- Docker
- GitHub Actions

### API
- 한국관광공사 TourAPI
